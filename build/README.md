# PrueEval

[![](https://img.shields.io/badge/License-GPL-green)](https://github.com/Lampese/PureEval/blob/main/LICENSE) ![](https://img.shields.io/badge/version-v2.2-red) [![](https://img.shields.io/badge/npm-pureeval-blue.svg)](https://www.npmjs.com/pureeval)

PureEval 因 [VoxelGeometry](https://github.com/CAIMEOX/VoxelGeometry) 项目而生，意在打造一个精巧而强大的 JavaScript 函数式工具包。

PureEval 具有以下特点：
<img align="right" src="https://raw.githubusercontent.com/PureEval/PureEval/main/logo.svg" height="150px" alt="logo">

- 支持对任意参数函数的柯里化和复变。
- 大量可以参与函数式编程的实用函数。
- 支持懒惰求值的列表（coming soon）。
- 拥有可用的 IO、Maybe、Either、Task 等抽象结构。
- 拥有一些实用工具如 optics、stateMachine 等。
- 体积小、代码精简，嵌入成本极低。

# Build

```bash
git clone https://github.com/PureEval/PureEval.git
cd PureEval
npm install
npm run build
```
